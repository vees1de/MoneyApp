# Feature: corporate-university

## Как понял модуль
Модуль внутреннего университета (Soft Skills) с направлениями, группами, тренерами, обратной связью и сертификатами.

## Что делает модуль
- Управляет направлениями обучения и внутренними программами.
- Формирует группы/потоки и расписание занятий.
- Поддерживает работу внутреннего тренера: участники, материалы, оценка.
- Собирает фидбек и итоговые оценки курса.

## Что бы я предпринял
- Завести справочник направлений (6 базовых направлений из ТЗ).
- Выделить сущности потока/группы/сессии для календарной интеграции.
- Добавить форму обратной связи и средний рейтинг курса.
- Подготовить выдачу сертификатов по критериям completion + pass.

## Вопросы для уточнения
- Направления фиксированы или редактируются администратором?
- Нужны ли очные/онлайн/смешанные форматы на уровне модели?
- Какой минимальный набор метрик качества курса нужен руководителю?
