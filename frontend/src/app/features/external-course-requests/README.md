# Feature: external-course-requests

## Как понял модуль
Это workflow заявок на внешние курсы с бюджетными ограничениями, документами и согласованием.

## Что делает модуль
- Создает заявку (ссылка, стоимость, сроки, программа, обоснование).
- Ведет жизненный цикл заявки и историю статусов.
- Проверяет бюджетные лимиты и полноту обязательных документов.
- Передает заявку в модуль согласований и календарной проверки.

## Что бы я предпринял
- Формализовать `ExternalCourseRequest` и схему валидации полей.
- Добавить проверки: бюджет подразделения, дубль заявки, корректность сроков.
- Сделать timeline статусов с комментариями согласующих.
- Реализовать пакетный просмотр заявок для HR/L&D.

## Вопросы для уточнения
- Какие документы обязательны для каждого типа курса?
- Как действуем при превышении бюджета: запрет или эскалация?
- Нужна ли возможность редактирования после отправки на согласование?
