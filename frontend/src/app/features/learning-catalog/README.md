# Feature: learning-catalog

## Как понял модуль
Каталог курсов — единая точка подбора релевантных программ (внутренних и внешних), чтобы снизить когнитивную перегрузку сотрудников.

## Что делает модуль
- Показывает список и карточки курсов с фильтрами и поиском.
- Разделяет курсы по типу: внутренний/внешний/обязательный.
- Показывает метаданные: длительность, формат, провайдер, стоимость, дедлайн.
- Позволяет перейти в назначение курса или создание заявки на внешний курс.

## Что бы я предпринял
- Реализовать фильтры: направление, формат, длительность, стоимость, обязательность.
- Добавить сортировку и быстрые пресеты (рекомендовано, дедлайн скоро, обязательные).
- Выделить `CatalogFacade` и кэш выдачи каталога.
- Подготовить контракт для рекомендаций в будущем (без реализации ML на MVP).
