# Feature: reports-analytics

## Как понял модуль
Модуль аналитики дает сквозную traceability-отчетность: от заявки до влияния на бюджет и прогресс подразделений.

## Что делает модуль
- Показывает дашборды по прохождению, обязательному обучению, срокам и бюджетам.
- Строит сквозные отчеты по процессу заявки и согласования.
- Экспортирует отчеты в Excel/PDF.
- Выдает срезы по подразделениям, ролям, периодам, направлениям.

## Что бы я предпринял
- Зафиксировать набор KPI MVP и их формулы.
- Подготовить витрину данных для стабильной производительности отчетов.
- Реализовать аудит качества данных (пропуски, несогласованность статусов).
- Сделать role-based доступ к отчетам и фильтрам.

## Вопросы для уточнения

- Нужны ли план/факт по бюджету на уровне подразделений?
- Какой период хранения исторических отчетов?
