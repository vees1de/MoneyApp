# Dashboard Test Role

## Route
`/dashboard/test-role`
`/playground`

## Purpose
Single playground route with all available widgets for layout checks.
Includes a YouGile agile widget based on board summary data.

## Widgets on page
- Team overview
- Upcoming events
- Jira summary
- Course requests
- Current learning
- Recommendations
- Quick actions
- My requests
- Work activity
- YouGile agile

## Notes
- No role switcher logic here.
- Real role separation lives in `/dashboard/{role}` pages.
