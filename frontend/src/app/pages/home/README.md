# Page: Home

## Route
`/home`

## Назначение
Защищенная домашняя страница пользователя после логина. На основе ролей и фич показывает доступные рабочие столы и быстрые переходы.

## Зависимости
### Features
- `auth`
- `notifications`

### Entities
- `user`

## Импорты (концептуально)
```ts
import { AuthStateService } from '@core/auth/auth-state.service';
import { UserRole, AppFeature } from '@core/auth/auth.types';
```

## Что происходит на странице
- Читает текущего пользователя, роли и доступные фичи из auth-state.
- Показывает виджеты перехода на релевантные разделы.
- Показывает блок последних действий.
