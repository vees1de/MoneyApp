# Page: My Learning List

## Route

`/my-learning`

## Что реализовано

- Список моих enrollments через `GET /api/v1/enrollments/my`.
- Отображение названия курса, статуса и дедлайна.
- Минималистичный список с фокусом на ключевом действии.
- Переход в детали конкретного enrollment: `/learning/:enrollmentId`.
